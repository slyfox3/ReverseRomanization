# How to Use?

By clicking around on the digital pool webpage, this extension shows the original names original names of the players when their romanized names is known to the buildt-in lookup table.

# Installation
1. Download this extension `.zip` file at [HERE](https://github.com/slyfox3/ReverseRomanization/archive/refs/heads/main.zip), and unzip.
2. Follow [these steps](https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world?hl=zh-tw#load-unpacked) to load the extension from the unzipped folder.

# How to Contribute?
Help maintain this [lookup table](https://github.com/slyfox3/ReverseRomanization/blob/main/name_mapping.js) together, and send a pull request!
